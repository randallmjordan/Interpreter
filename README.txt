README

Vandall Interpreter is a class project for CS3320, System Level Programming, at Georgia State University in the Summer 2013. The idea is to create a scripting language and program an interpreter for said language. 

SYSTEM REQUIREMENTS

Java JDK 1.6.0_14 or higher

FILE MANIFEST

README.txt
Interpreter.java
sum.v
sort.v
powers.v
compare.v
kitty.v

OPERATING INSTRUCTIONS

User will need to compile Interpreter.java
Once compiled and .class file is created, user can do the following, from command prompt, to read scripts
java Interpreter file
example:
	using included files
	java Interpreter sort.v
